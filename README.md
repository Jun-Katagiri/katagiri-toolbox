default structure generated by `create-turbo@latest`.\
It will serve as a development toolbox where multiple small apps and
shared packages are managed in one place.

## ğŸ¯ Purpose of This Monorepo

This repository is intended to evolve into a **toolbox** containing
multiple small utilities:

-   Sequence Runner (sequence-based step display tool)
-   Regex Editor (pattern + logic editor)
-   Simon 9-panel game
-   IR document checker
-   Other small tools useful for translation, UI testing, and general
    workflows

Each new tool will be placed under `apps/` as its own Next.js
application.

Shared logic will gradually move to `packages/` (`ui`, `db`, `utils`,
`types`).
